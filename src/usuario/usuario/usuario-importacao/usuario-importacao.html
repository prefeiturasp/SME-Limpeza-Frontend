IMPORTAÇÃO DE USUÁRIOS

Arquivo de Dados

Certifique-se que seu arquivo possua a estrutura de dados necessária para a importação e o carregue abaixo para importar os usuários.

  • Formato: CSV
  • Cabeçalho: id_origem;origem_chave;nome;email;cargo_ue;url_nomeacao
  • Separador: ponto e vírgula (;)
  • Linha Zero: cabeçalho
  • Case Sensitive: Sim
  • Chave: E-mail
  • Id Origem: 1 (SME) | 2 (DRE) | 3 (UE)
  • Origem Chave: nome da DRE ou código da UE
  • Cargo da UE: FT | FS | R
