IMPORTAÇÃO DE AMBIENTES DE UNIDADES ESCOLARES

Arquivo de Dados

Certifique-se que seu arquivo possua a estrutura de dados necessária para a importação e o carregue abaixo para importar os ambientes das unidades escolares.

  • Formato: CSV
  • Cabeçalho: descricao;ambiente_geral_descricao;codigo_ue;tipo;area
  • Obrigatoriedade: Todos
  • Separador: ponto e vírgula (;)
  • Linha Zero: Cabeçalho
  • Case Sensitive: Sim
  • A descrição do Ambiente geral deve ser exatamente igual a cadastrada no sistema.
  • O código da Unidade Escolar deve ser exatamente igual ao cadastrado no sistema.
  • O valor da área deve utilizar ponto como separador decimal.
  • Tipo: 'I' para interno e 'E' para externo
